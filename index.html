Política de Privacidade

Sudoku Master — Última atualização: 01 de abril de 2026

A DESENVOLVIMENTO E CONSULTORIA EM TI ENN LTDA (CNPJ 63.721.800/0001-11), doravante denominada "Codeenn", é a responsável pelo aplicativo Sudoku Master. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários.

1. Dados que Coletamos

O Sudoku Master coleta o mínimo de dados necessários para seu funcionamento:

  • Dados de compra in-app: Quando você realiza uma compra premium, os dados da transação são processados pelo Google Play Billing e pelo RevenueCat (nosso provedor de gestão de compras). Isso inclui identificadores anônimos de transação, tipo de produto adquirido e status da compra.
  • Identificadores do dispositivo: Um identificador anônimo pode ser gerado pelo RevenueCat para gerenciar suas compras e restaurar assinaturas entre dispositivos.
  • Dados de desempenho do app: Informações técnicas como versão do app, sistema operacional e relatórios de erro podem ser coletados para melhorar a qualidade do aplicativo.

O Sudoku Master NÃO coleta: nome, e-mail, telefone, localização, fotos, contatos ou qualquer dado pessoal identificável.

2. Como Usamos os Dados

  • Gerenciamento de compras: Para processar, validar e restaurar suas compras premium através do RevenueCat.
  • Melhoria do app: Dados de desempenho são utilizados exclusivamente para identificar e corrigir problemas técnicos.

3. Compartilhamento de Dados

Compartilhamos dados apenas com os seguintes provedores de serviço, estritamente necessários para o funcionamento do app:

  • RevenueCat, Inc. — Gestão de compras in-app e entitlements. Política de Privacidade do RevenueCat.
  • Google Play Billing — Processamento de pagamentos. Política de Privacidade do Google.

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins de marketing ou publicidade.

4. Armazenamento Local

O Sudoku Master armazena dados de jogo (progresso, estatísticas, preferências) localmente no seu dispositivo usando localStorage. Esses dados não são transmitidos para servidores externos e podem ser removidos ao desinstalar o aplicativo ou limpar os dados do app.

5. Segurança

Todas as comunicações entre o app e os servidores de terceiros (RevenueCat, Google Play) são realizadas por conexões criptografadas (HTTPS/TLS). Adotamos medidas técnicas razoáveis para proteger os dados processados.

6. Crianças

O Sudoku Master não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores. Se tomarmos conhecimento de que dados de uma criança foram coletados inadvertidamente, os excluiremos prontamente.

7. Seus Direitos

Você tem direito a:

  • Solicitar informações sobre os dados que processamos;
  • Solicitar a exclusão dos seus dados;
  • Revogar seu consentimento a qualquer momento;
  • Restaurar suas compras em outro dispositivo.

Para exercer seus direitos, entre em contato pelo e-mail: contato@codeenn.com.

8. Exclusão de Dados

Para solicitar a exclusão dos seus dados:

  • Envie um e-mail para contato@codeenn.com com o assunto "Exclusão de Dados — Sudoku Master";
  • Dados locais podem ser excluídos a qualquer momento limpando os dados do app nas configurações do seu dispositivo;
  • Dados de compra no RevenueCat serão excluídos conforme a política de retenção do provedor.

9. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. A data da última atualização será sempre indicada no topo do documento. Recomendamos que você revise esta política regularmente.
